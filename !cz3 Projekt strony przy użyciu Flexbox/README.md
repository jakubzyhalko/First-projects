\# Strona Flexbox



\## Opis



Projekt strony przy użyciu Flexboxa, HTML, SASS oraz Javascriptu. Strona posiada nawigację, kontakt oraz karty. Strona jest responsywna dla urządzeń mobilnych



\## Technologie:



&#x20;   - HTML

&#x20;   - Flexbox

&#x20;   - SASS

&#x20;   -JavaScript
